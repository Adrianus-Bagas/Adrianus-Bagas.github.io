Proyek yang Dikerjakan

Analisis Sentimen pada Komentar Terhadap Kebijakan Perjalanan Domestik yang Dikelompokkan Menggunakan Metode Self-Organizing Maps

Selama masa pandemi COVID-19, aktivitas transportasi diatur dalam kebijakan perjalanan domestik yang mengharuskan penumpang untuk memenuhi beberapa persyaratan bila ingin bepergian. Kebijakan yang dibuat selalu berubah-ubah sehingga membuat masyarakat mengeluh. Proyek ini merupakan tugas akhir saya saat menempuh pendidikan Sarjana Matematika dimana saya melakukan text mining menggunakan Self-Organizing Maps untuk mengelompokkan opini masyarakat dan Indonesian Sentiment Lexicon untuk mengetahui sentimen setiap kelompok opini. Penjelasan lebih lanjut untuk proyek ini dapat dilihat pada draft tugas akhir, poster, dan video serta source code dapat diakses disini.
